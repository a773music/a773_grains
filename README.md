Clock multiplier/divider (mult_div)
-
https://www.instagram.com/p/BBI0L4PmHXQ/?taken-by=a773music

Pot 1/Input 1  
Multiplication

Pot 2/Input 2  
Division

Pot 3  
CCW: Simple ratios (1,2,4,8,16...)  
Middle pos: Stopped  
CW: Prime ratios (1,3,5,7,11,13...)

Input 3  
Clock in

Out  
Clock out
